# cvesee

a package vulnerability info gathering tool for CLI

## planned capabilities

- generate and verify URLs for Nist NVD and Canonical CVE trackers
- gather and parse data from above sites
- output clean list of vulnerable packages, with filtering by Ubuntu release
- allow configuration of custom info sources
- access and parse data from GHSA
- present statuses like "CVE number assigned", "reporting CNAs", "patch available (and where)", etc
- access custom package repositories and cross reference to CVE(s) of interest
- surely more...
